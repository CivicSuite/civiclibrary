# CivicLibrary Implementation Plan

## v0.1.0

Ship the runtime foundation only:

- FastAPI service and health endpoint.
- Public sample UI.
- Deterministic library policy Q&A, program/event answer, collection-metadata reference search, and collection-development guidance helpers.
- Docs, tests, browser QA evidence, release gate, and package build.

## Later

- Isolated deployment profile.
- Patron-privacy-gated connector design.
- Librarian review queue.
- Optional local LLM prompt layer through CivicCore.
