Shipping v0.1.1

Library support that protects patron privacy.

CivicLibrary helps library staff draft cited policy answers, answer program questions, search collection metadata without patron records, and prepare collection-development guidance for librarian review.

For Staff

Draft policy answers, program replies, reference suggestions, and collection guidance that librarians verify before use.

For IT

FastAPI runtime pinned to civiccore==0.3.0, deterministic tests, and no ILS writes.

Privacy Boundary

Patron records are out of scope; v0.1.1 works only with public policies, program information, and collection metadata.

Architecture

Boundaries

CivicLibrary v0.1.1 does not access patron records, integrate with an ILS, replace professional reference service, provide legal advice, call live LLMs, or ship library connector runtime.
