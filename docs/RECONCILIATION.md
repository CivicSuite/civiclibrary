# CivicLibrary Reconciliation

The canonical catalog defines CivicLibrary as library policy and reference support: library policy Q&A, program and event Q&A for patrons, reference assistance over collection metadata rather than patron records, collection-development guidance, and accessibility support for library materials.

v0.1.0 intentionally ships deterministic foundations only. It does not access patron records, connect to an ILS, replace professional reference service, provide legal advice, call live LLMs, or ship library connector runtime.
